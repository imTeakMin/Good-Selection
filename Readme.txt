[실행 방법]

1. Android Studio 실행 ( Android Studio 설치 시 애뮬레이터 설치해야됨, 만약 애뮬레이터 없으면 갤럭시 기기 연결하여 사용)
2. 왼쪽 상단 File 메뉴에서 Open... 클릭
3. 압축 해제한 폴더 중 "GoodSelection" 폴더를 선택하고
4. 프로젝트가 열리면 자동으로 Gradle Sync가 실행됩니다.
   문제가 생기면 상단 메뉴에서 File > Sync Project with Gradle Files 클릭
5. 중앙 상단에 Run 'app'을 클릭하여 연결된 에뮬레이터 또는 실제 기기에서 실행

[Test 방법]

- 회원가입 기능 사용
1. 로그인 화면에서 계정 생성 기능 사용을 위해 회원가입 버튼 클릭
2. 이메일 형식의 아이디와 비밀번호 작성 후 가입완료 버튼을 누르면 로그인 화면으로 이동

- 로그인 기능 사용
3. 회원가입한 아이디와 비밀번호를 입력하여 로그인 버튼을 눌러 메인 화면으로 이동

- Run Ladder 기능 사용
4. 메인화면에서 Run Ladder 버튼 클릭 후 선택지 개수 입력 
   (최대 6까지 가능 그 보다 큰 수 입력시 아래에 최대 6개 가능 문구 뜸)
5. 확인 버튼을 누르면 시작 지점 입력 칸과 도착 지점 입력 칸이 화면에 뜨고 
   선택지에 빈칸없이 모두 입력 후 게임 시작 버튼 클릭
6. 사다리 타기가 끝나고 아래에 결과 보기 버튼 클릭해서 결과 화면을 이동
7. 결과 화면에 메인으로 이동 버튼 클릭 시 메인 화면으로 이동
   광고 보기 버튼 클릭 시 광고 화면으로 이동
8. 광고 보기 버튼을 클릭했다면 광고가 화면에 뜨고 광고가 끝나면 나가기 버튼 클릭
9. 나가기 버튼 클릭 시 메인 화면으로 이동하고 상단에 포인트가 20 포인트 증가되어있음

- Run Roulette 기능 사용
10. 메인화면에서 Run Roulette 버튼 클릭 후 선택지 개수 입력
    (최대 8까지 가능 그 보다 큰 수 입력시 아래에 최대 8개 가능 문구 뜸)
11. 확인 버튼을 누르면 선택지 개수 입력만큼 입력 칸이 화면에 뜨고
     선택지에 빈칸없이 모두 입력 후 게임 시작 버튼 클릭
12. 시작 버튼을 클릭하여 룰렛을 실행
13. 메인으로 이동 버튼 클릭 시 메인 화면으로 이동
     광고 보기 버튼 클릭 시 광고 화면으로 이동
14. 광고 보기 버튼을 클릭했다면 광고가 화면에 뜨고 광고가 끝나면 나가기 버튼 클릭
15. 나가기 버튼 클릭 시 메인 화면으로 이동하고 상단에 포인트가 20 포인트 증가되어있음

- Logout 기능 사용
16. 메인화면에서 Logout 버튼 클릭 후 로그아웃 화면으로 이동
17. 로그아웃 화면에서 확인 버튼 클릭 시 현재 정보 저장하고 로그인 화면으로 이동
     취소 버튼 클릭 시 다시 메인 화면으로 이동

- Membership Cancel 기능 사용
18. 메인화면에서 Membership Cacnel 버튼 클릭 후 계정 삭제 화면으로 이동
19. 계정 삭제 화면에서 확인 버튼 클릭 시 firebase에서 계정 정보 삭제 후 로그인 화면으로 이동 ( 해당 계정으로 로그인 불가)
     취소 버튼 클릭 시 로그인 화면으로 이동 ( 삭제 한 계정으로 로그인 시도 시 로그인 실패 메시지 뜸)

- Purchase 기능 사용

현재 새로 만든 계정은 포인트가 적기 때문에 테스트를 위한 포인트가 많은 계정을 알려드리겠습니다.
아래의 로그인 아이디와 비밀번호로 로그인해서 Purchase, Storage 기능을 사용하면 됩니다.

아이디 : sunday@gmail.com
비밀번호 : qwer1234!

20. 위의 계정으로 메인 화면으로 이동 후 Purchase 버튼 클릭 후 상품 구매 화면으로 이동
21. 구매하고 싶은 상품 아래에 구매하기 버튼을 클릭하여 상품을 구매
22. 포인트가 부족하면 구매하기 버튼을 눌러도 포인트가 감소하지 않고 Storage에 추가 안됨
     위의 계정은 포인트가 충분하므로 구매하기 버튼을 누르면 사용자의 포인트가 감소하고 Storage에 추가됨
23. 애뮬레이터 화면 위쪽에 뒤로 가기 버튼을 클릭하거나 ctrl+shift+B 클릭하여 메인 화면으로 이동

- Storage 기능 사용
24. 메인 화면에서 Storage 버튼 클릭 후 보관함 화면으로 이동
25. 방금 Purchase 기능을 통해 구매 한 상품들이 Storage에 추가 되어있음
26. 상품 아래에 사용하기 버튼을 클릭하면 Storage에서 사라짐
27. 애뮬레이터 화면 위쪽에 뒤로 가기 버튼을 클릭하거나 ctrl+shift+B 클릭하여 메인 화면으로 이동
